# 📊 Tableau Dashboard — Superstore

An end-to-end business intelligence dashboard project focused on executive KPI reporting and data-driven decision-making.
---

## 📌 Project Overview  
This project presents an interactive Tableau dashboard built using the Superstore dataset to analyze business performance across sales, profit, and orders.

The goal is to demonstrate how KPI dashboards can help organizations track performance and identify key business trends quickly.

---

## 🎯 Objectives  
- Analyze overall business performance using KPIs  
- Identify sales and profit trends across categories  
- Compare performance across product segments  
- Build an interactive and easy-to-use Tableau dashboard  

---

## 📊 Dashboard Features  
- KPI Cards: Total Sales, Total Profit, Total Orders  
- Category Filter: Furniture, Office Supplies, Technology  
- Interactive visuals for business exploration  
- Clean layout for executive-level reporting
