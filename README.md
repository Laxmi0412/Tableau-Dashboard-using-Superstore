# 📊 Tableau Dashboard — Superstore

A Tableau KPI dashboard using the Superstore dataset showing sales, profit, and order insights with interactive visualization.

---

## 📌 Overview
This project demonstrates how to build a KPI dashboard in Tableau to analyze business performance using the Superstore dataset.

---

## 📊 Dashboard Preview

![Dashboard](tableau-kpi-dashboard.png)

---

## 📊 Dashboard Components
- KPI Cards — Total Sales, Total Profit, Total Orders
- Category Filter — Furniture, Office Supplies, Technology
- Clean layout for quick business insights

---

## 📈 Key Insights
- Total Sales: $2.29M  
- Total Profit: $286K  
- Total Orders: 9,994  
- Technology category has highest profit  
- Office Supplies has highest order volume
